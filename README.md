##  프로그램 실행 방법 

1. programing 폴더를 visual studio로 프로젝트를 불러옵니다.
2. sin 파일을 더블클릭 하고 컴파일 합니다.


>프로젝트에대하여 보다 자세한 사항은 hwp 보고서 파일을 참고해 주시기 바랍니다.
