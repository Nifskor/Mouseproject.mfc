[고양이가 된 쥐]

◎ 메인 화면
‑ 게임 제목 Static text
‑ 개발자 Static text
‑ 시작 Button
‑ 종료 Button → OnOK()

◎ 시작 Button
‑ 게임 설정 Modal Dialog 생성
‑ (보류) 난이도 선택 Radio 버튼 (원 사라지는 타이밍)
‑ (보류) 원 색깔 지정
‑ (보류) 도형 선택
‑ 확인 버튼
‑ 취소 버튼

◎ 메뉴
‑ 게임 방법
‑ 정보 (개발자 이름, 개발 기간 등)
‑ 나가기

◎ 게임 화면
‑ 점수 표시
‑ 남은 시간 표시
‑ (보류) 팝업 메뉴 (종료)

◎ 게임 시작
‑ 무작위 위치 원 생성
‑ 일정 시간 뒤 원 사라짐
‑ 마우스 닿음으로 원 사라짐
‑ 마우스 닿음으로 점수 +1
‑ 게임 진행 시간 Timer 설정
‑ 시간이 끝나면 게임 종료

◎ 게임 결과
‑ 결과 창 Modal Dialog 표시
‑ 점수 표시
‑ (보류) 선택했던 난이도 표시
‑ (보류) 점수와 난이도를 반영해 랭크 표기 (S, A, B, C)
‑ 다시하기, 메인화면, 종료 Button
